# be rich 가속기 로또 예측 프로젝트 [be rich 加速器 乐透预测项目]

1 프로젝트 개요 [项目概况]
이 프로젝트는 시계열 데이터 분석과 딥러닝을 활용하여 로또 번호의 통계적 패턴을 분석하는 연구용 실험실입니다. [本项目是利用时序数据分析和深度学习，分析乐透号码统计模式的研究用实验室。]
교수님의 BMAD 가이드라인을 준수하여 재현 가능한 데이터 파이프라인을 구축하였습니다. [遵守教授的 BMAD 指南，构建了可再现的数据流水线。]

2 주요 특징 [主要特点]
데이터 레이어는 공식 API를 통해 실시간으로 당첨 번호를 동기화합니다. [数据层通过官方 API 实时同步中奖号码。]
모델 레이어는 머신러닝 베이스라인과 Bi LSTM 기반 딥러닝 모델의 성능을 비교합니다. [模型层对比了机器学习基准和基于 Bi-LSTM 的深度学习模型的性能。]
결과 레이어는 백테스트를 통해 모델의 예측 정확도를 시각화하고 보고서를 생성합니다. [结果层通过回测对模型的预测准确度进行可视化并生成报告。]

3 시작하기 [开始使用]
실험을 실행하려면 먼저 필요한 라이브러리를 설치해야 합니다. [若要运行实验，首先需要安装必要的库。]
그 후 scripts 폴더에 있는 파이프라인 스크립트를 실행하십시오. [之后请运行 scripts 文件夹中的流水线脚本。]

명령어 예시 [命令 예시]
python3 scripts run_pipeline.py [运行流水线命令]

4 프로젝트 구조 [项目结构]
bmad output 프로젝트 설계 및 요구사항 정의서가 들어있습니다. [_bmad-output 文件夹包含项目设计及需求定义书。]
src 데이터 처리 및 모델 핵심 로직이 들어있습니다. [src 文件夹包含数据处理及模型核心逻辑。]
scripts 전체 프로세스를 한 번에 실행하는 스크립트가 들어있습니다. [scripts 文件夹包含一次性执行全流程的脚本。]
reports 실험 결과 및 성능 분석 보고서가 들어있습니다. [reports 文件夹包含实验结果及性能分析报告。]
config 프로젝트의 모든 경로와 파라미터를 설정하는 파일입니다. [config 文件夹是项目所有路径和参数的配置文件。]

5 주의사항 [注意事项]
이 프로젝트는 연구 목적으로 제작되었으며 실제 당첨을 보장하지 않습니다. [本项目为研究目的制作，不保证实际中奖。]
원본 데이터인 data raw 폴더의 파일은 수정하지 않는 것을 원칙으로 합니다. [原则上不修改 data/raw 文件夹中的原始数据文件。]
