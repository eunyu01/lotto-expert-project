# 요구사항 정의서 [PRD 需求文档]

1 개요 [개요]
본 프로젝트는 로또 당첨 데이터를 수집하여 딥러닝 모델로 다음 회차 번호의 확률 분포를 예측합니다. [本项目收集乐透中奖数据，利用深度学习模型预测下一期号码的概率分布。]
이를 전문가 전략으로 필터링하여 제공하는 것을 목표로 합니다. [目标是通过专家策略进行过滤后提供。]

2 기능적 요구사항 [功能要求]
공식 API를 통한 데이터 최신화 기능을 제공합니다. [提供通过官方 API 进行数据更新的功能。]
연번, 홀짝, 대소 등 형태학적 특징 추출 기능을 제공합니다. [提供连号、奇偶、大小等形态学特征提取功能。]
머신러닝 로지스틱 회귀를 이용한 베이스라인 구축을 포함합니다. [包括利用机器学习逻辑回归构建基准。]
PyTorch Bi LSTM 기반 딥러닝 모델 학습 및 예측 기능을 포함합니다. [包括基于 PyTorch Bi-LSTM 的深度学习模型训练和预测功能。]
모델 간 성능 비교 시각화 보고서 생성 기능을 제공합니다. [提供模型间性能对比可视化报告生成功能。]

3 비기능적 요구사항 [非功能要求]
모든 파일 경로는 상대 경로를 사용하여 재현성을 보장합니다. [所有文件路径均使用相对路径，以确保再现性。]
설정 파일을 통해 모든 파라미터를 관리합니다. [通过配置文件管理所有参数。]
코드 변경 시 스크립트를 통해 자동 실행이 가능해야 합니다. [代码更改时，必须能够通过脚本进行自动执行。]

4 데이터 요구사항 [数据要求]
데이터 소스는 dhlottery API를 사용합니다. [数据来源使用 dhlottery API。]
데이터 형식은 CSV 파일을 사용합니다. [数据格式使用 CSV 文件。]
타겟 데이터는 다음 회차 당첨 번호입니다. [目标数据是下一期中奖号码。]
