# 아키텍처 및 작업 설계 [架构及任务设计]

1 시스템 아키텍처 [系统架构]
데이터 레이어는 API 수집을 통해 원본 데이터를 저장하고 전처리를 수행합니다. [数据层通过 API 采集保存原始数据并进行预处理。]
모델 레이어는 전통적 머신러닝 베이스라인과 신경망 기반 딥러닝 모델로 구성됩니다. [模型层由传统机器学习基准和基于神经网络的深度学习模型组成。]
실행 레이어는 scripts 폴더 내 진입점을 통해 전체 파이프라인을 구동합니다. [执行层通过 scripts 文件夹内的入口点驱动整个流水线。]
결과물은 학습된 모델 파일과 성능 지표 데이터를 포함합니다. [成果物包括训练好的模型文件和性能指标数据。]
