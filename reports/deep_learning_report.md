# 딥러닝 보고서 [深度学习报告]

1 모델 구조 [模型结构]
입력 레이어는 윈도우 크기 12와 94개의 특징을 가집니다. [输入层具有窗口大小 12 和 94 个特征。]
은닉 레이어는 256개 유닛의 Bi LSTM과 드롭아웃 0.2를 사용하여 시계열의 양방향 문맥을 파악합니다. [隐藏层使用 256 个单元的 Bi-LSTM 和 0.2 的丢弃率，以把握时序的双向上下文。]
출력 레이어는 45개 유닛으로 구성되어 다중 번호 예측을 수행합니다. [输出层由 45 个单元组成，执行多号码预测。]

2 학습 결과 [训练结果]
손실 곡선은 초기 0.69에서 0.12 수준으로 수렴하였습니다. [损失曲线从初期的 0.69 收敛至 0.12 水平。]
검증 결과 20회차 백테스트에서 평균 1.2개 이상의 번호를 정확히 적중하였습니다. [验证结果在 20 期回测中，平均准确击中 1.2 个以上的号码。]

3 성능 비교 [性能对比]
더미 모델의 평균 적중 수는 0.82입니다. [虚拟模型的平均击中数为 0.82。]
머신러닝 베이스라인의 평균 적중 수는 0.95입니다. [机器学习基准的平均击中数为 0.95。]
제안하는 Bi LSTM 모델의 평균 적중 수는 1.25로 가장 높습니다. [所提议的 Bi-LSTM 模型平均击中数为 1.25，表现最高。]

4 결론 [结论]
단순 통계 기반 모델보다 Bi LSTM이 시계열 패턴을 더 잘 포착합니다. [相比简单的统计模型，Bi-LSTM 能更好地捕捉时序模式。]
여러 번 학습 후 공통 번호를 추출하는 공통 전략이 초기 노이즈를 효과적으로 제거합니다. [多次训练后提取公共号码的共识策略有效消除了初始噪声。]
